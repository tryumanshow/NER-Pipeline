
- 데이터 디렉터리
    ```
    ⎿ data
        ⎿ completed
        ⎿ preprocessed
        ⎿ raw
        ⎿ unk
    ```

- completed: 모델 학습 및 검증에 사용되는 최종 데이터
- preprocessed: completed 데이터를 만들기 위한 전 단계
- raw: raw dataset
- unk: unk 토큰이 등장하는 경우에 대한 기록  


- 용량으로 인하여, 아래 세 데이터는 Git에 업로드하지 않습니다.  
    1. completed
    2. preprocessed
    3. raw
